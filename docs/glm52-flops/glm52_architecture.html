DSA / NSA Sparse Attention

GLM-5.2 exposes DSA indexer fields in the config. In SGLang, that routes the model to the dsa backend instead of ordinary full-context MLA when the backend is left to defaults.

The DSA indexer changes the long-context scaling story. The attention page counts the selected-key scan math and separately calls out that indexer/top-k work is additional overhead.

Speculative Decoding

The checkpoint has num_nextn_predict_layers = 1. A plain non-speculative forward does not use it. In our SGLang serving setup, --speculative-algorithm EAGLE loads the same checkpoint as the draft model and remaps the nextn layer into the draft path.
