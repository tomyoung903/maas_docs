Scope

The constant contains matrix multiplications whose work is fixed for one token: MLA projections in all 78 layers, DSA indexer query/key/gate setup in 21 refresh layers, three dense MLPs, and 75 active MoE blocks. It excludes the vocabulary head and all work whose size depends on context length.

The indexer candidate-score scan is not missing. It compares a new index query against cached candidate keys, so its work changes with context length. Count it with the selected-key MLA scan on the DSA + MLA page, not in a linear constant.
