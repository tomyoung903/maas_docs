How To Read This Page

The source report is valuable as an execution trace: it shows the actual rank-0 module path and the shapes produced by a 10,000-token prefill. It is not a trustworthy model-accounting engine. Shared module references caused recursive rows, every decoder layer was printed separately, embedding lookup was treated like dense arithmetic, and all resident MoE experts were charged as active work.

This page therefore joins two sources without mixing their meanings: runtime shapes come from the instrumented run, while parameter state comes from the audited GLM-5.2 architecture inventory. Repeat badges expand each distinct row back to the complete target stack.

Expand The 78 Layers
